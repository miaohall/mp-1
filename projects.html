Predicting Phishing Scams Based on URL Features

In this project my teammates and I developed a Python based analysis using machine learning algorithms. We used K Nearest Neighbors and Random Forest classification models with Scikit-Learn to classify URLs as phishing or non-phishing. We conducted statistical analyses, such as t-tests and correlation studies using Matplotlib, SciPy, and NumPy. We created visualizations including feature importance plots, box plots, scatter plots, and histograms using Matplotlib, NumPy and Seaborn.
